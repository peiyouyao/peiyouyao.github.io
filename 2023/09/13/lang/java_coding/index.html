总结

  • 对于对象流 (Stream<T>), toArray() 可以将流元素收集到 Object[], 或者指定类型的数组中.
  • 对于基本类型流 (IntStream、LongStream、DoubleStream), toArray() 会返回对应类型的基本数组 (如 int[]、long[]、double[]).
  • 使用 toArray(IntFunction<A[]> generator) 可以更方便地生成特定类型的数组, 并避免类型转换的麻烦.
